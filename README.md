



<h1> Simple chat application </h1>

<h2> What is there to notice in this demo-app ? </h2>

<ul> 
 
<li>Animations</li>
<li>Firebase Auth</li>
<li>Firebase Firestore</li>
<li>Firebase Storage</li>
<li>Lottie Animations</li>
